W katalogu znajduje się jeden program:
- num9.py - stosuje on wszystkie możliwe z czterech omówionych metod szukania miejsca zerowego dla dwóch zadanych funkcji oraz dla jednej, będącej uproszczeniem algorytmu dla funkcji o dwukrotnym pierwiastku. Dla każdej z trzech badanych funkcji wypisywane są w każdej iteracji wartości różnicy między oszacowaną wartością x a dokładnym miejscem zerowym.

Katalog zawiera również Makefile z komendą:
- make run - uruchamia program num9
